This is a build log: a running record of small, real software projects, built step by step, with the working code included — plus a growing set of explained LeetCode solutions.

Every tutorial follows the same shape — what we're building, what it takes, and the exact steps to get there — so you can follow along and end up with something that actually runs, not just a concept.

No fluff, no "10 tips" posts. If it's here, it's because it was built and tested.

## What you'll find

- Full project walkthroughs (CLI tools, scripts, small apps, browser extensions)
- Explanations of *why* each step works, not just what to type
- Working code you can copy, run, and adapt
- LeetCode solutions with the approach explained, not just the answer

## Who this is for

Anyone comfortable with the basics of at least one programming language who wants to learn by building something real, or who's prepping for coding interviews and wants explanations, not just answers.

Got a project idea or a problem you'd like explained? Reach out — link in the footer.
